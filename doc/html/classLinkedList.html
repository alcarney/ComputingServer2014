Detailed Description

template<class Type>
class LinkedList< Type >

A template class that describes a Linked List.

This template manages a series of nodes that are linked via a pointer to the next node in the list and the previous node in the list. This class currently supports adding nodes to the end of the list, looping through the list from a given index and deleting its contents.
